API DOCUMENTATION

Replit Server Monitoring Service to prevent your server dying by sending HTTP Request to your keep_alive/server.

API ROUTES

Method: GET

/Docs -> documentation page for this api. (no params.)

Method: GET

/addmonitor -> returns a json response that contains the api's response if success or failed.

/addmonitor Params

url -> link to your replit's server.

/addmonitor?url="replit-server-url"
